Resume

В сферу моей деятельности входит создани сайтов и интернет магазинов с нуля и на различных CMS и CMF платформах. Разрабатываю проекты на: Werdpress, Woocommerce, OpenCart, Vue. Адаптация Тем и CMS-шаблонов.

B многолетней практике, в основном, работал под эсклюзивный индивидуальный дизайн. Без исползования лишних, не нужных, перегружающих систему файлов. JavaScript, jQuery использую по степени необходимости. В работе с хостингом применяю FileZilla. GitHub - опыт имею, так же в связке с VS Code.

WordPress/WooComerce - установка, создание темы, верстка, работа с плагинами и виджетами, разворачивание на хостинге. WooComerce - WordPress plugin интернет магазина.

OpenCart - популярный движек интернет магазина. Установка, работа с темами и плагинами, изменение дизайна(css + изменение структуры кода), разворачивание на хостинге.

CSS фреймверки(типа бутстрап) + JS анимационные эффекты(Animate.css).

HTML5/CSS3 - могу обучать специфике и тонкостям начинающих и малоопытных работников, имею опыт.
