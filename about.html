About

В сферу деятельности входит создание интернет-проектов с нуля на различных CMS-платформах: веб-сайты, лендинги, интернет-магазины и т.д. Среди разработанных мной проектов: Werdpress, Woocommerce, OpenCart, Vue. Адаптация тем и шаблонов CMS.

B многолетней практике, в основном, работал под эсклюзивный индивидуальный дизайн. Без исползования лишних, не нужных, перегружающих систему файлов. JavaScript, jQuery использую по степени необходимости. В работе с хостингом применяю FileZilla. GitHub - опыт имею, так же в связке с VS Code.

WordPress/WooComerce - установка, создание темы, верстка, работа с плагинами и виджетами, разворачивание на хостинге. WooComerce — плагин WordPress для интернет-магазинов.

OpenCart - популярный движек интернет магазина. Установка, работа с темами и плагинами, изменение дизайна(css + изменение структуры кода), разворачивание на хостинге.

CSS фреймверки(типа бутстрап) + JS анимационные эффекты(Animate.css, AOS Animate).

Образование - Харківський національний університет міського господарства (ХНУМГ) імені О.М.Бекетова.
